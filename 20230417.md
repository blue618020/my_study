## Python
- Explorer(파일 아이콘) : 저장한 코드 파일 불러오기
- 코드 실행 : 오른쪽 상단 위 세모 아이콘(재생) 또는 단축키 ctrl + F5 
- TERMINAL(터미널, 또는 쉘) 창에서 실행 확인 가능
- 주석 : ctrl + /
- 줄 바꾸기 : alt + 방향키
- 문자형
  - ""
  - ''
  - " '' "
  - ' "" '
  - """ """ or ''' ''' (\n과 같은 기능으로 간단하게 줄바꿈 가능)
- 숫자형
  - 정수(int형) : 1, 2, 3 ...
  - 실수(flort형) : 1.1, 2.3 .... 
  - 허수(complex형) : -1, -2 ....
- 연산자
  - \+ 더하기 연산자
  - \- 빼기 연산자
  - \* 곱하기 연산자
  - / 나누기 연산자
  - \*\* 제곱 연산자
  - // 정수 나누기(몫) 연산자
  - % 나머지 연산자
- 이스케이프 문자
  - \n 줄바꿈
  - \t 탭
  - \\" 쌍따음표
  - \\' 따음표
