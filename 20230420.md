## whlie 반복문
- 조건이 Ture일 경우에 코드를 계속 반복
- 특정 조건 또는 시점에 충족할 때까지 반복할 때 사용하기 좋음 

### 대입 연산자
- +=
- -=
- *=
- /=
- **=
- //=
- %=
- n += 1 은 n = n + 1 라는 의미
- 산술연산자 다 가능
- 문자열에서도 적용 가능

### break
- 반복문을 즉시 종료함
- if문으로 종료 조건을 걸어서 같이 사용함

### continue
- while 반복문의 제일 처음으로 돌아감
